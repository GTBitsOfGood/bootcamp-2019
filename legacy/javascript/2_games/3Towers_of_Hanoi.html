3Towers_of_Hanoi

Instructions and source code | Test code

Click "Start new game" to start a new game of Towers of Hanoi. Use the "Select" and "Drop buttons to move disks between towers. Your goal is to move all the disks to the rightmost side.
Rules:
  • Use the "Select" and "Drop" buttons.
  • You may only move one disk at a time.
  • You must never allow a bigger disk to go on top of a smaller disk.
  • All three disks start on the leftmost tower.
  • Game is won when all three disks are on the rightmost tower.
